03 — Typography
Type System
Two fonts only. Sora for all body and UI text. Space Mono for labels, code, metadata, and technical identifiers.
Primary Font — Sora
Aa Bb Cc
All headings, body text, buttons, UI copy. Weights: 300 Light, 400 Regular, 600 SemiBold, 700 Bold.

05 — Brand Voice & Messaging
How We Talk
AutoScale's voice is friendly, confident, and human. We speak like a smart friend who happens to be an expert — not a corporate robot.
Personality Traits
Friendly — Warm and approachable, like talking to a colleague
Confident — We know our stuff, but not arrogant
Clear — No jargon unless the audience is technical
Empowering — We help you grow, we're your partner
Innovative — We stay ahead, we're excited about the future
Core Messages
Save time → "Let automation handle the routine. Focus on growth."
Save money → "Stop paying for tasks a bot can do in seconds."
Scale smart → "Grow without the overhead."
AI for everyone → "Enterprise-level AI, startup-friendly pricing."
Results → "Real businesses, real results — automated."

08 — Rules for GitHub Pages Websites
Website Standards
Every AutoScale GitHub Pages site must follow these rules so all sites feel like one cohesive brand.
Mandatory HTML Head Template
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1, viewport-fit=cover">
<meta name="theme-color" content="#060d1a">
<meta name="apple-mobile-web-app-capable" content="yes">
<!-- Tab favicon -->
<link rel="icon" type="image/jpeg" href="https://raw.githubusercontent.com/richard36901-source/Home-page/main/logo.jpg">
<link rel="apple-touch-icon" href="https://raw.githubusercontent.com/richard36901-source/Home-page/main/logo.jpg">
<!-- Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Space+Mono:wght@400;700&family=Sora:wght@300;400;600;700&display=swap" rel="stylesheet">
Mandatory CSS Variables
--bg: #060d1a;
--glass: rgba(12,26,56,0.55);
--border: rgba(80,140,255,0.13);
--text: #e8f0ff;
--muted: #6a85b6;
--accent: #3a8ef6;
--accent2: #1dffd4;
--radius: 18px;
Background Layers (Always Use)
Layer 1: Solid background #060d1a
Layer 2: Radial gradient overlays (blue top-left, teal top-right)
Layer 3: 40×40px grid pattern at 4% opacity
Cards: Glass + backdrop-filter: blur(12px)
Component Checklist — Every Page Must Have
Header with logo (72×72px, border-radius 18px)
Brand name gradient text "AutoScale"
Tagline "אוטומציות חכמות" in Space Mono
Section labels in Space Mono, uppercase, 10.5px, 2px tracking
Glass cards with backdrop-filter blur
Tab favicon pointing to logo.jpg URL
RTL direction for Hebrew pages (dir="rtl")
Mobile-responsive (max-width: 560px centered)
Footer with "עודכן לאחרונה" date in Space Mono
Body background grid + radial gradients
✅ Always Do
Use the exact CSS variables defined above
Use Sora + Space Mono — no other fonts
Dark background on all AutoScale pages
Add the favicon URL to every page head
Use glass cards with backdrop-filter
Add radial gradient background layers
Center content max-width 560px (mobile-first)
❌ Never Do
Don't use white or light backgrounds
Don't use Arial, Inter, Roboto, or system fonts
Don't use purple gradients (generic AI look)
Don't change the color variables
Don't use solid borders without transparency
Don't skip the favicon link tag
Don't hardcode pixel sizes without CSS variables
